# 主题学习说明

## 主题列表
- **Technology** - 科技主题
- **Economy** - 经济主题
- **Environment** - 环境主题
- **Health** - 健康主题
- **Education** - 教育主题
- **Politics** - 政治主题

## 学习周期结构

每个主题下包含多个学习周期（Week01, Week02...），每个周期为7天：
- **Day 01-03**：基础阶段 - 建立主题基础词汇和基本语法结构
- **Day 04-07**：进阶阶段 - 深化理解，提高复杂文本处理能力

## 使用方法

### 开始新主题学习
1. 选择或创建主题目录
2. 创建新的Week目录（如Week01）
3. 使用模板开始记录每日学习内容

### 学习记录格式
- 每日记录：`day01.md`, `day02.md` ... `day07.md`
- 周期总结：`summary.md`
- 使用模板：`Templates/daily_learning_template.md`

### 学习内容要求
1. **困难单词**：标注音标、词性、释义、例句
2. **高频短语**：标注释义、用法、例句
3. **长难句式语法**：分析句式结构、语法要点
4. **当日学习文章**：记录文章信息、内容、分析
5. **长难句或专业文章选段**：深入分析难点
6. **学习笔记**：记录重点、难点、复习建议、反思

## 学习目标

通过7天周期的系统学习，达到以下目标：
- 掌握主题相关的基础和进阶词汇
- 理解并应用复杂语法结构
- 提高专业文章阅读能力
- 建立系统的学习笔记体系
